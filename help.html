Instructions

Main Menu:

Top Bar:

Title and time: In the center of the Top Bar, the program's name and the current time is displayed. This cannot be changed by the user.

Note Creation Button: On the leftmost side of the top bar is a Plus icon (+) that, when pressed, presents the user with a window to create a new note.

Note Deletion Button: On the rightmost side of the top bar is a Trashcan icon (ï€”) that, when pressed, presents the user with a window to delete a note.

If the user wants to create a new note, they can do so by clicking the Note Creation Button.

If the user wants to delete a note, they can do so by clicking the Note Deletion Button.

Note Display:

This segment of the Program contains all the notes that have been created. Each note consists of 3 properties:

Title: The title of the note, which is editable by the user when in Editing mode.

Date: The date and time that the note was created. This cannot be normally changed by the User.

Image: The image that is displayed in the note. This can be changed by the user when in Editing mode.

To select a note, all you have to do is simply click a note in the Display.

Button Bar:

This segment of the program, which is present in all modes, contains buttons that perform different actions depending on the mode.

Edit Note Button: The Edit Note Button, that when pressed with a note highlited, will put the user in Edit mode for the selected note

(When in Delete mode) Delete Note Button: The Delete Note Button, that when pressed with a note highlited, will permanently delete the selected note.

Settings Button: The Settings button, when pressed, presents the user with a window that allows the user to change the program's settings

Help Button: The Help button, when pressed, opens this webpage for the user to search for help.

Modes:

There are three modes in this program:

Normal: The default mode the program always opens in. This mode allows the user to create and edit notes, and can be reopened in edit mode by closing the note.

Edit: This mode allows the user to edit a note. This mode can be opened in normal mode by selecting a note, and then clicking the Edit Selected Note button.

Delete: This mode allows the user to Delete selected notes. This mode is opened by clicking the Trashcan icon in the top bar.
